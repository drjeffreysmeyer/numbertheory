Definition 1.1.4. Units.

We will say an element \(a \in R\) is multiplicatively invertible if it has a multiplicative inverse, i.e. if there exists an element \(b \in R\) such that \(ab=1\text{.}\) When it is clear from context, we will simply say \(a\) is invertible, or that \(a\) is a unit.
