Proposition 3.4.5. The phi function of a prime power.

If \(p\in \Z\) is prime and \(k\in \Z_{>0}\text{,}\) then \(\phi(p^k)=p^k-p^{k-1}\text{.}\)
🔗
🔗

Hint.

Try a direct proof. It is sometimes helpful to think about about least resides modulo \(p^k\) as a grid. Write down the first \(p\) many least residues modulo \(p^k\) then go to a new line. Repeat this process for the remaining rows until you have a \(p\) by \(p^{k-1}\) grid. Think about the columns and rows that contain elements relatively prime to \(p^k\text{.}\) Try for a few small primes first. Remember, though, examples never constitute a proof. You may find Definition 3.1.1, Definition 2.2.2, and Definition 2.3.4 useful.
